# WorkTimeTracker - プレミアム工数管理ツール

WorkTimeTrackerは、直感的な操作とモダンなデザインを兼ね備えた、個人・プロジェクト向けの工数管理（勤怠管理）アプリケーションです。

*(ここにアプリの画像を配置予定)*

## ✨ 主な特徴

- **マルチプロジェクト管理**: 複数の案件やプロジェクトを個別に管理。
- **モダンなBento Grid UI**: 情報を整理して表示する美しく使いやすいインターフェース。
- **動的な進捗追跡**: 月間目標時間に対する現在の稼働状況をリアルタイムで視覚化。
- **柔軟なエクスポート**: 勤怠報告にそのまま使えるCSV出力と、バックアップ用のJSON出力に対応。
- **オフライン・ローカル保存**: Electronの機能を利用し、データはすべてローカルPCに安全に保存されます。

## 🚀 使用方法

### 1. プロジェクトの作成と切り替え
- 画面左上のドロップダウンからプロジェクトを切り替えます。
- 「+」ボタンを押すと、新しいプロジェクト（案件）を追加できます。

### 2. 勤務時間の入力
- カレンダーの日付をクリックして、入力ダイアログを開きます。
- **開始・終了時間**: デフォルト値を設定から変更可能です。
- **休憩**: 分単位で入力します。
- **出社フラグ**: 特徴的なアイコンで出社日（オフィス勤務）をマーキングできます。

### 3. 目標管理と統計
- 右側のサイドバーで、当月の合計稼働時間と目標時間（最小/最大）の乖離を確認できます。
- 日別の稼働推移が棒グラフで表示され、ペース配分を一目で把握できます。

### 4. 設定
- 歯車アイコンから、アプリのテーマカラー、標準勤務時間、月間目標時間をカスタマイズできます。

## 🛠 テクニカルスタック

- **Frontend**: React, Vite, Lucide React (Icons), Recharts (Graphs)
- **Runtime**: Electron
- **Styling**: Vanilla CSS (Premium Glassmorphism Design)
- **Persistence**: Node.js File System via Electron IPC

```bash

npm install

# 開発モード（Electron）の起動
npm run electron:dev

npm run electron:build
```
